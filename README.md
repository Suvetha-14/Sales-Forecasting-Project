# 📊 Sales Forecasting using Linear Regression

## 📌 Project Overview
Businesses often struggle to estimate future sales based on past performance.  
This project builds a **Linear Regression model** to predict future sales using historical sales data and visualizes insights through a **Power BI dashboard**.

---

## 🎯 Objective
Develop a regression model that can **forecast future sales trends** based on historical data.

---

## 📂 Dataset
The dataset contains the following fields:

- **Date** – Transaction date  
- **Product** – Product name  
- **Quantity** – Number of items sold  
- **Revenue** – Total revenue generated  

---

## ⚙️ Project Workflow

1. **Data Collection**
   - Loaded sales dataset from CSV file

2. **Data Preprocessing**
   - Checked data types
   - Handled missing/null values
   - Cleaned the dataset

3. **Exploratory Data Analysis (EDA)**
   - Visualized sales trends
   - Analyzed product performance

4. **Machine Learning Model**
   - Applied **Linear Regression**
   - Trained model to predict future sales

5. **Model Evaluation**
   - Compared **Actual vs Predicted Sales**

6. **Data Visualization**
   - Built an **interactive Power BI Dashboard**

---

## 📊 Power BI Dashboard
The Power BI dashboard provides insights such as:

- Sales trend over time
- Product-wise sales analysis
- Actual vs predicted sales comparison
- Forecasted sales insights
